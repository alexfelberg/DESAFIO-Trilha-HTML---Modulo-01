DESAFIO: Criar uma página Web utilizando as tags aprendidas


Trilha HTML: Módulo 01

A ideia deste desafio é elaborar um website, utilizando uma estrutura bem básica, utilizando tags aprendidas durante as aulas práticas até então.

III. Descrição das tags:

  1. Tag h1, h2, h3, h4, h5 e h6:
    - Define a hierarquia dos títulos da página. A tag h1 tem importância maior que a tag h2 e assim sucessivamente.

  2. Tag p:
    - Acrescenta um parágrafo na página.

  3. Tag mark:
    - Acrescenta um destaque no texto selecionado.

  4. Tag small:
    - Reduz a fonte do texto selecionado.

  5. Tag i:
    - Acrescenta o itálico no texto selecionado.

  6. Tag u:
    - Acrescenta o sublinhado no texto selecionado.

  7. Tag strong:
    - Acrescenta uma grande importância ao texto selecionado.

  8. Tag ol:
    - Acrescenta uma lista ordenada à página.

  9. Tag ul:
    - Acrescenta uma lista não ordenada à página.

  10. Tag li:
    - Define o item da lista.

  11. Tag a:
    - Acrescenta um hyperlink ao texto selecionado.

  12. Tag hr:
    - Acrescenta uma linha horizontal à página, com o objetivo de separar textos distintos.

  13. Tag sub:
    - Define o texto selecionado como subscrito.

  14. Tag sup:
    - Define o texto selecionado como sobrescrito.

  15. Tag blockquote:
    - É utilizada para realização de citações no texto.

  16. Tag font:
    - É utilizada para definir as especificações da fonte no texto, entretanto ela é suportada no HTML4. No HTML5 não é mais utilizada

  17. Tag del:
    - É utilizada para substituir uma palavra um texto excluído por outro.

  18. Tag br:
    - É utilizada para inserir quebra de linhas no texto.

  19. Tag abbr:
    - É utilizada para inserir o significado de uma abreviatura no texto em HTML.
